## Project Scope

The objective of this project is to identify whether a news article is REAL or FAKE using Machine Learning techniques. The system analyzes the news text and predicts its authenticity based on patterns learned from training data.

## Technologies Used

- Python
- Google Colab
- Pandas
- Scikit-Learn
- TF-IDF Vectorizer
- Logistic Regression

## Project Description

Fake news has become a major issue in today's digital world. This project uses Natural Language Processing (NLP) and Machine Learning to classify news articles as REAL or FAKE. The text data is converted into numerical features using TF-IDF Vectorization, and a Logistic Regression model is trained to perform the classification.

## Features

- Detects fake and real news articles
- Uses Machine Learning for classification
- Simple and easy-to-understand implementation
- Can be extended with larger datasets
